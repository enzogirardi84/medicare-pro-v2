## 2. Mapa GPS — 3 iteraciones (bugs de Streamlit Cloud)

### `views/dashboard.py` — Mapa de visitas (líneas 304-374)

#### Iteración 1 (commit `184a197`): st.map()
- **Problema**: `st.map()` usa MapLibre demo tiles que NO tienen calles ni lugares. Solo puntos.
- **Solución**: Reemplazar por PyDeck con CARTO dark-matter tiles.

#### Iteración 2 (commit `184a197`): PyDeck + CARTO
- **Problema**: PyDeck con CARTO tiles requiere conexión a `basemaps.cartocdn.com`. En Streamlit Cloud las tiles no se cargaban correctamente (sin calles visibles).
- **Solución**: Reemplazar por Leaflet + OpenStreetMap tiles vía HTML embebido.

#### Iteración 3 (commit `5600421`, definitivo): Leaflet + OpenStreetMap
```python
_html_map = f"""
<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" />
<script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"></script>
...
var _map = L.map('map').setView([{_lat_center}, {_lon_center}], 13);
L.tileLayer('https://{{s}}.tile.openstreetmap.org/{{z}}/{{x}}/{{y}}.png', {{
    attribution: '&copy; OpenStreetMap contributors',
    maxZoom: 19,
}}).addTo(_map);
{_markers_html}
"""
st.components.v1.html(_html_map, height=500)
```
- **Cómo funciona**: Usa `st.components.v1.html()` para embeber HTML/JS con Leaflet. Leaflet carga tiles de OpenStreetMap (calles, rutas, localidades, etc.). Los marcadores GPS se agregan con `L.marker().bindPopup()`.
- **Sin API key**, sin dependencias nuevas.
- **Popups**: muestran paciente, tipo y fecha al hacer click en cada marcador.
- **Importante**: usa `escape()` de `html` para prevenir XSS en nombres de paciente.
